Comment jouer à Qui est-ce ?

 

Peut importe votre âge, si vous aimez les jeux de déduction, vous allez adorer le jeu de société "WooaMye ?"!

Il vous suffit juste de trouver le personnage de votre adversaire ( sélectionné aléatoirement).
En utilisant les questions prédéfinies dont la seule réponse sera OUI ou NON !

Exemple de question : Est ce que le personnage a une barbe ?

- Si la réponse à la question est OUI dans ce cas le joueur peut sélectionner tous les personnages qui n'ont pas de barbe.

-- Si la réponse est NON alors il pourra sélectionner les personnages qui ont une barbe.

Lorsque vous pensez avoir identifié le personnage vous pouvez le sélectionner et valider votre choix.

-Si la réponse donnée est fausse vous perdez la partie et la bonne carte vous sera montrée.
-Si vous choisissez la bonne carte vous avez gagné.
